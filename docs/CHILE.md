# Especificación Global de Lenguaje Chileno (UX Writing + Copywriting)

## Objetivo

Toda la aplicación debe sentirse diseñada para usuarios chilenos.

No solamente la landing page.

Incluye:

- Landing
- Catálogo
- Buscador
- Carrito
- Checkout
- Cuenta de usuario
- Dashboard
- Historial
- Soporte
- Emails
- Notificaciones
- Errores
- Modales
- Tooltips
- FAQ
- Blog
- LMS
- Centro de ayuda
- Panel administrativo

La experiencia debe sentirse:

- Local
- Cercana
- Moderna
- Profesional
- Confiable
- Gamer
- Fácil de entender

---

# Principio Principal

La aplicación NO debe parecer traducida desde inglés.

Debe parecer escrita originalmente por una empresa chilena.

Proceed to checkout
